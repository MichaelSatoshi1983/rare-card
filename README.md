这是我在26年3月，在TRAE Friends现场的workshop里，两小时做出来的小项目。

当时的目标很简单，做一个可以快速整理罕见病患者关键信息的小工具，它不承担完整病历的功能，也不应该假装替代医生诊断，只是希望在紧急的时候把这些信息清楚的放在一起。

后面我又把它补了一些，现在它已经变得简单能用。**当然，它还算不上一个正儿八经的产品，它还很差，不过，至少从一个workshop demo变成一个我继续愿意打磨的小工具，这就够了。**

# 它可以做什么

- 按照步骤填写信息
- 生成可以打印的急诊卡
- 支持多种尺寸
- 支持AI生成
- ...等等等

# 为什么做这个

罕见病的信息很复杂，但是在发病现场或者急诊的时候，没那么多时间解释，很多时候真正需要的不是常见的说明，而是把几个关键点讲清楚，这个项目就是围绕这些信息做的。

# 本地部署

你得需要一个pnpm。

```bash
pnpm install
pnpm dev
# Build
pnpm build
# Test
pnpm test
```

# 值得注意的是

**这个工具生成的内容只能用于急诊沟通参考，不构成任何医疗建议。如果你真的要打印、导出、或者随身带着，请务必让患者本人、有能力的监护人或者是医生来核实内容，特别是AI辅助生成的字段，更应该当作草稿，而不是最终的医学结论。**
